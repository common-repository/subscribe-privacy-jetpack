Adds agreement for privacy policy to mail subscriptions. Needs the Jetpack plugin with the use of subscription form.

== Description ==
When the user subscribe to a newsletter, he must agree with terms of service and privacy policy.
This rule is included in the GDPR (General Data Protection Regulation).

This plugin is intended to complete the subscription form of the Jetpack plugin.
After being activated, all subscription forms will show a checkbox with a link to the WP privacy policy.
Please, check that you set a privacy policy page in the Settings > Privacy page.

= Requirements =

* Jetpack
* WordPress version required by Jetpack.

== Installation ==
 
1. Install and activate Subscribe privacy for Jetpack
2. If you don't have it already, install [Jetpack](https://wordpress.org/plugins/jetpack/), activate and connect it to WordPress.com if you're prompted to do it.
3. Add an e-mail subscribe form (widget for example)
4. You should see the policy agreement !
